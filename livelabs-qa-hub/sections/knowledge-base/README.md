# Knowledge Base

Purpose: provide a NotebookLM-style QA source intake lane for files, links, research notes, and reviewed decision records.

Current prototype scope:

- Local seeded source notes.
- Admin-only note creation.
- Review pipeline counters for tracked, reviewed, and pending sources.
- No external file upload, parsing, embeddings, or network processing.

Production direction:

- Add source ingestion with file/link metadata.
- Summarize and classify sources before promotion.
- Require human review before accepted knowledge influences QA rules, requirements, or reports.
- Link accepted knowledge to Watchdog decisions and LiveLabs TMS requirements.
