# QA Watchdog

Purpose: provide a Datadog-like quality operations surface for LiveLabs QA alerts, logs, severity, ownership, status, and evidence.

Initial capabilities:

- Monitor active alerts across platform, LiveStack, content, automation, analytics, and sprint operations.
- Create manual QA flags when a reviewer finds a problem.
- Track severity, source, owner, and status.
- Link future records to Jira, CI/CD artifacts, screenshots, logs, and reports.

MVP candidate: yes.
