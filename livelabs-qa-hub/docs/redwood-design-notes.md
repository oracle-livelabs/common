- Oracle JET UI rules from the active `redwood-creator` skill.

## Design Choices

- Use Oracle Sans copied into the project so the prototype is portable.
- Use Oracle Red `#C74634`, Oracle Bark `#312D2A`, neutral surfaces, and functional link blue `#00688C`.
- Keep the first draft as an operational app, not a marketing landing page.
- Use a left-side app navigation rail for hub sections.
- Keep cards flat with restrained borders and small radii.
- Keep admin-only actions visible only when the active role can use them.

## Known Prototype Limits

- This is Redwood-aligned CSS, not a certified Oracle JET implementation.
- Production UI should map controls to Oracle JET/Core Pack components if Oracle JET becomes the approved stack.
