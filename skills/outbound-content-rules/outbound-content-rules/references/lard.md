# Lard Factor


Show some examples of sentences whose word count we could reduce, and show how to re-word them. 

Calculate **lard factor** as a function of "word count before" (w_0) and "word count after" (w_1):

\[
\text{LF} = \frac{w_0 - w_1}{w_0} \times 100\%
\]

Display the output like this:

   ## Example %d

   Original sentence:
   %s

   Re-worded sentence:
   %s

   Word count before: %d words
   Word count after: %d words
   Lard factor (LF): %d%%

How much do you think we could reduce the word count without losing any meaning?
