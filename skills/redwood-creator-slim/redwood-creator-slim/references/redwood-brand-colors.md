# Redwood Brand Colors

## Local source artifacts

- Primary source summary: extracted into this portable reference file; the original PDF is not bundled in the slim package.
- Supporting note: extracted into this portable reference file; the original local research note is not bundled.

Use the bundled rules below by default. Re-check an approved Oracle source outside the package only when exact tonal ramps, print values, or texture treatments matter.

## Signature colors

- Oracle Red: `#C74634`
- Oracle Bark: `#312D2A`

Start Oracle-branded work from these anchor colors before introducing pillar-specific accents.

## Secondary and pillar-anchor colors

- Brand Yellow: `#F0CC71`
- Brand Green: `#5F7D4F`
- Beige 40: `#E9E1CA`
- Slate 60: `#A9BBBC`
- Database / Slate 100: `#697778`
- HCM / Rose 100: `#A36472`
- Dev Tools / Pebble 100: `#6F757E`
- Finance / Teal 100: `#4F7D7B`
- CX / Plum 110: `#796087`
- GBU / Lilac 100: `#6B7494`
- SCM / Sienna 100: `#AA643B`
- NetSuite / Ocean 100: `#437C94`
- Brand Neutral 100: `#7A736E`
- OCI / Pine 100: `#4C825C`

Use pillar colors when the content clearly maps to that business area. Otherwise stay closer to the signature palette and neutrals.

## Functional colors

- Link color on light backgrounds: Link Blue 120 `#00688C`
- Link color on dark backgrounds: Accent Yellow 50 `#F0CC71`
- Text on dark backgrounds: Text White `#FFFFFF`
- Text on light backgrounds: Text Neutral 190 `#161513`

Treat these four rules as hard constraints unless the user supplies a stronger Oracle source.

## Usage rules

- Use HEX values for digital work.
- Use CMYK and PMS only when a print deliverable explicitly needs them.
- Do not expect CMYK or PMS to match on-screen HEX exactly.
- Default Oracle-branded outputs to Oracle Red, Oracle Bark, the neutral family, and the documented functional text/link colors.
- Use pillar-specific light-to-dark ramps only when the underlying content benefits from that pillar association.
- Treat pillar textures as branded accent surfaces, not generic background patterns.

## Flag as non-compliant

- Arbitrary accent colors where a documented Redwood value should be used
- Light-background text that is not aligned with `#161513`
- Dark-background text that is not aligned with `#FFFFFF`
- Link colors that ignore the light/dark rule
- Pillar colors used without a clear contextual reason
