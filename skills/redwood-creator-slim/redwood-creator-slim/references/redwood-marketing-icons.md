# Redwood Marketing Icons

## Local source artifacts

- Primary source summary: extracted into this portable reference file; the original presentation is not bundled in the slim package.
- Supporting note: extracted into this portable reference file; the original local research note is not bundled.

Use this summary for normal routing and compliance decisions. Re-check an approved Oracle source outside the package only when exact icon lookup is required.

## What the library is

- Standard Oracle Redwood marketing icon library
- Cover title: `Redwood Marketing Icon Library`
- Version observed: `v4.7, January 2026`
- Owner text on cover: `Brand Experience Creative`
- Visual style: monochrome outline pictograms arranged in labeled slide grids

This is a marketing and illustration library, not the primary source of application control icons.

## Best-fit use cases

- Presentations
- Landing-page illustration
- Diagrams
- Branded collateral
- Social graphics
- Non-interactive explanatory artwork

## Do not use as the primary source for

- Oracle JET control icons
- Navigation glyphs
- Form-field affordances
- Table, chart, or dashboard control chrome
- Product UI iconography that should follow JET behavior

## Practical rules

- Prefer this library when the user needs branded pictograms, illustrative concepts, or collateral art.
- Treat the naming as fuzzy-match input rather than a strict schema: labels mix hyphens, underscores, and occasional inconsistencies.
- If both marketing illustration and app UI appear in the same artifact, keep this library confined to the illustration layer.

## Flag as non-compliant

- Marketing pictograms used as app control icons
- Marketing pictograms used as navigation glyphs
- Marketing pictograms replacing JET component affordances
