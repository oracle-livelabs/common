# Lard Factor

Show examples of sentences whose word count can be reduced, and show how to reword them.

Calculate lard factor as a function of word count before (`w0`) and word count after (`w1`):

`LF = ((w0 - w1) / w0) * 100%`

Display the output like this:

## Example %d

Original sentence:
%s

Re-worded sentence:
%s

Word count before: %d words
Word count after: %d words
Lard factor (LF): %d%%

Ask how much the word count can be reduced without losing meaning.
