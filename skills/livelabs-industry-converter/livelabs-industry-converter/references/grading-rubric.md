# Grading Rubric

Grade in order. Stop at the first layer that fails.

## Layer 1: Validator Readiness

Check:
- LiveLabs file structure,
- markdown mechanics,
- spelling, punctuation, and grammar,
- manifest resolution,
- launch-file coherence,
- no duplicated intro metadata or repeated validator-repair blocks.

Block a pass if any validator-equivalent rule fails.

## Layer 2: Source Fidelity

Check:
- source sections are still present,
- source task count is preserved,
- source step count is materially preserved,
- source screenshots and images are still present or intentionally mapped,
- introductions, explanations, and conclusions are not unnecessarily shortened,
- generic labs remain close to the source where no domain rewrite was required,
- source restoration did not create duplicate headings, duplicate time lines, or duplicate objective sections,
- validator repairs did not introduce repeated body text, repeated code blocks, or repeated task instructions,
- side-by-side review confirms fidelity beyond line counts and validator status.

Fail this layer if the conversion dropped content, collapsed walkthroughs, or rewrote generic material without a clear need.

## Layer 3: Prose Quality

Check:
- active voice,
- strong verbs,
- concise sentences,
- low filler,
- no contractions in final scored prose,
- no em dashes,
- no unsupported claims.

## Layer 4: Structure And Flow

Check:
- preserved lab sequence,
- smooth progression,
- readable paragraph length,
- clear task framing,
- non-nested lists unless the source format requires otherwise.

## Layer 5: Domain Authenticity

Check:
- believable industry entities and roles,
- consistent terminology,
- realistic statuses and IDs,
- examples that fit the target audience,
- no stray source-domain artifacts.

## Layer 6: Technical Defensibility

Check:
- preserved architecture,
- valid commands, queries, and object names,
- coherent sample data and outputs,
- defensible statements,
- no unresolved conjecture presented as fact.

## Final Bar

Do not deliver the workshop as complete unless every layer passes at `A` quality.

If a file fails, revise the file. If a sentence fails, revise the sentence. Do not explain away defects that can be fixed.
